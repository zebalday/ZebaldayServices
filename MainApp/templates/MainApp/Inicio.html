He creado este simple sitio web con la intención de proporcionar servicios multimedia útiles para mi y otras personas. Hasta el momento sólo he añadido 2 servicios: Generar códigos QR a partir de data ingresada, y descargar videos de Youtube a partir de una url válida. A continuación se explica resumidamente en qué consiste cada uno de los módulos utilizados.

QRcode

Un código de respuesta rápida (Quick Responde Code) es un código pictográfico bidimensional que se utiliza por su rápida legibilidad y su capacidad de almacenamiento comparativamente grande. El código consta de módulos negros dispuestos en un patrón cuadrado sobre un fondo blanco. La información codificada puede estar compuesta por cualquier tipo de datos (por ejemplo, símbolos binarios, alfanuméricos o Kanji).

Revisa la documentación aquí.

PyTube

Pytube es una biblioteca Python genuina, liviana y libre de dependencias (y una utilidad de línea de comandos) para descargar videos de YouTube. pytube es una biblioteca ligera escrita en Python. No tiene dependencias de terceros y pretende ser altamente confiable. Pytube también facilita la canalización, lo que le permite especificar funciones de devolución de llamada para diferentes eventos de descarga, como en progreso o completo. Además, pytube incluye una utilidad de línea de comandos que le permite descargar videos directamente desde la terminal.

Revisa la documentación aquí.
